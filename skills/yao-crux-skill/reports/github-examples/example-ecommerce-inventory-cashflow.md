## 主要矛盾判断过程

这个案例不能只看GMV增长或库存积压，而要看哪一个矛盾同时解释断货、现金紧、清仓难和投放争议。

第一性原理追问：如果只能改变一个经营机制，改变什么能让增长不再吞噬现金？

### 看得见的问题

- 看得见的问题：GMV增长，但现金流更紧。
- 看得见的问题：热销SKU断货，错过流量窗口。
- 看得见的问题：旧款和低动销尺码占用库存。
- 看得见的问题：采购、投放和财务互相拉扯。

### 上升一层：找看不见的根部变量

上升一层看，热销款断货和旧款积压是同一个系统问题的两面：团队还没有用SKU分层和现金周转来约束采购、投放和清仓。

- 看不见的根部变量：SKU分层、补货纪律和现金周转没有成为统一决策规则。
- 看不见的根部变量：投放预算没有和可售库存、毛利和补货周期联动。
- 看不见的根部变量：清仓、补货和上新没有在同一张经营账上取舍。

### 为什么不是先处理表面问题

如果只继续加投放，可能放大断货和现金压力；如果只清库存，可能牺牲热销增长。当前要先建立SKU分层和现金周转规则，让采购和投放都围绕同一套经营约束行动。

### 候选矛盾怎么排序

排序时看哪个矛盾最能同时影响GMV、现金、库存和团队协作。库存现金周转失衡比单独补货或清仓更上游。

## 主要矛盾（最关键的卡点）

- 主要矛盾：订单增长 与 库存现金周转失衡的冲突
- 主要方面（现在最影响局面的一侧）：库存现金周转失衡当前占支配地位
- 为什么先处理它：当前不是单纯卖得不够，也不是单纯库存太多，而是增长没有被SKU分层、补货纪律和现金周转约束住。这个矛盾不解决，GMV越高越可能吞噬现金。
- 什么情况说明判断要改：如果A类SKU断货率下降、低动销库存下降，但利润仍没有改善，主要矛盾可能转向价格体系、毛利结构或产品企划能力。

## 次要矛盾（先不主攻，但要盯住）

| 次要矛盾 | 为什么暂时不主攻 | 什么时候要处理 |
| --- | --- | --- |
| 低动销库存 与 现金回收需求的冲突 | 需要马上处理，但它要和A类补货放在同一张现金周转表里统一取舍。 | 如果低动销库存占比两周内没有下降，就要加大清仓或渠道分流。 |
| 热销款需求 与 补货周期滞后的冲突 | 它很重要，但要在SKU分层后优先保障A类SKU，否则补货资源仍会被分散。 | 如果A类SKU已明确但仍断货率超过10%，就升级补货机制。 |
| 投放增长目标 与 可售库存约束的冲突 | 投放规则要跟随库存分层调整，不宜单独追求GMV。 | 如果投放仍把预算打到库存不足SKU上，就升级投放治理。 |
| 新品上新节奏 与 历史库存消化的冲突 | 当前先稳住周转和现金，再处理上新节奏。 | 如果现金压力下降但新品命中率低，再升级产品企划问题。 |

### 哪些变化会影响这个估计

- 如果A类SKU断货率仍高于10%，建议下调行动后概率。
- 如果低动销库存下降但毛利明显恶化，需要重新平衡清仓策略。
- 如果供应商交期缩短到14天以内，可以上调增长恢复概率。

## 主要矛盾什么时候会转移

当前先控制库存现金周转；一旦SKU分层稳定，主要矛盾可能转向产品企划、供应链柔性或品牌价格体系。

| 阶段 | 可能主导的矛盾 | 触发条件 |
| --- | --- | --- |
| 当前60天 | 订单增长 与 库存现金周转失衡的冲突 | 增长越快，现金和库存压力越大。 |
| 周转改善后 | 产品企划节奏 与 市场需求变化的冲突 | 库存可控后，新品命中率成为增长瓶颈。 |
