# Artifact Design Profile

Artifact family: high-trust expert learning report plus tutorial packet.

## Visual Direction

- restrained editorial report
- compact evidence blocks
- readable tables with subtle borders
- topic-led headings
- no decorative gradients, glass panels, or repeated card grids

## Format Strategy

Markdown is the canonical source. DOCX, PDF, and HTML are generated exports.

HTML should feel like a readable research report:

- fixed left numbered menu
- centered report shell
- compact section anchors
- responsive table wrapping
- standalone CSS

DOCX/PDF should feel like formal internal learning documents:

- clear heading hierarchy
- readable tables
- no visible local paths
- no overflow-prone raw URLs inside cells
- print CSS that keeps content visible

## Density Rules

- Split wide tables.
- Move long source details to the appendix.
- Use one-page brief before deep sections.
- Put keyword cards in grouped sections or compact tables.
- Keep self-test and rubric easy to scan.
