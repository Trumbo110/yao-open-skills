## Layout Notes

The examples are intentionally kept with exported artifacts because this skill's value depends on multi-format report generation. The HTML examples show the fixed left-side navigation tree. The PDF examples are generated from print-safe HTML with the navigation removed before printing. The DOCX examples demonstrate table-width and overflow handling.
